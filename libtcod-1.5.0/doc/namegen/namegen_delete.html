=====================================
16.3 Destroying a generator
=====================================

 > 16.4 Retrieving available set names

TCODNamegen::destroy

To release the resources used by a name generator, you may call:

C++ : static void TCODNamegen::destroy(void)
C   : void TCOD_namegen_destroy(void)
Py  : namegen_destroy()

This will free all memory used by the generator. In order to generate a name again, you have to call TCODNamegen::parse again.
